Privilege Separation

Kernel Mode: Direct hardware access (RAM/CPU).
User Mode: Restricted area for apps (Bash/Web Servers). Apps use System Calls to talk to the Kernel.

👤

User Identity

Root: The superuser (UID 0).
/etc/passwd: The user registry.
Sudo: Temporary admin power with an audit trail (best practice).

⌨️

Power Commands

  • ls -la: Detailed file list
  • cd ..: Move up a level
  • cat: View file contents
  • man: Documentation
